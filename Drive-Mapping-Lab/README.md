# Drive Mapping Lab (GPO)

This project shows how I deployed automatic drive mappings based on department using Group Policy Preferences.

---

## 🧰 Tools & Technologies
- Group Policy Management  
- Active Directory  
- Windows Server 2019  
- Windows 10 Client

---

## 🛠️ What I Completed
### ✔ Created departmental shared folders  
### ✔ Created security groups for each department  
### ✔ Mapped drives using GPO Preferences  
Examples:
- HR → H:\  
- Helpdesk → S:\  
- IT → T:\  

### ✔ Linked GPO to Workstations OU  
### ✔ Confirmed mappings apply to correct users  
### ✔ Tested login to multiple domain accounts
