## What I Configured
- Minimum password length
- Maximum password age

- Complexity requirements
- Lockout / retry thresholds

## Skills Demonstrated
- Group Policy Management
- Domain-wide security baselines
- Linking & enforcing GPOs
- Testing using `gpupdate` and `rsop.msc`
