# Active Directory Lab

This project demonstrates my ability to deploy and administer a Windows Server 2019 Active Directory Domain Services (AD DS) environment.  
I built a fully functional domain called **adlab.local**, added client machines, and configured users, OUs, and security groups.

---

## 🧰 Tools & Technologies
- Windows Server 2019  
- Active Directory Domain Services (AD DS)  
- DNS  
- VirtualBox  
- Windows 10 (Client01)
